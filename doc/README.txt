Voici les étapes à suivre afin de faire fonctionner le projet :

1 - Importer le projet

2 - Lancer le projet en mode serveur

3 - Dans le dossier "client" du projet se trouve un fichier "testIntegration.html".
Ce fichier peut se trouver n'importe où sur votre ordinateur et même sur votre réseau.
Il fonctionnera tout de même (à noter que si vous le placez sur une autre machine de votre réseau,
Il vous faudra alors remplacer "localhost" par le nom de votre machine sur le réseau).
